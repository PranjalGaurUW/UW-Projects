# UW-Projects
Project contained in 'Exploratory Data Analysis - Cricket' folder is contains exploratory data analysis on a sample of cricket matches data scraped from the web. The analysis includes - analyzing behavior of batting and bowling across innings and teams in a T20 game, generating a 20-ball highlights of second innings of each game (that shows turning points of the game- such as wickets/outs, fours, sixes etc.), clustering games based on scores, optimizing Duckworth-Lewis method to generate our own based on sample data.

Project contained in 'Quantitative Data Analysis Project - R' folder is based on Regression analysis. Performed quantitative analysis on Canada population survey data by implementing regression models (linear and ordinal logistic regression) in R to study the effects of social factors on covid vaccine acceptance likelihood.
